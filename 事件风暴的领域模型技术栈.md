### ️ 最终版技术选型表（UI 框架更新为 Ant Design Vue）

| 架构层级 | 核心技术 | 核心职责与选型理由 |
| :--- | :--- | :--- |
| **应用外壳** | **Tauri v2.0** | 跨平台容器（Win/Mac/Linux/移动端）。极小安装包体积，系统级 API 调用能力。 |
| **前端框架** | **Vue 3 + TypeScript** | 界面渲染核心。配合 Vite 提供极速开发体验，TS 保证类型安全。 |
| **状态管理** | **Pinia** | Vue 3 官方推荐状态管理库，管理画布节点数据、用户设置、协同状态等。 |
| **图形引擎** | **AntV X6** | 核心画布渲染。负责节点拖拽、连线、缩放、贝塞尔曲线等图形交互。 |
| **UI 组件库** | **Ant Design Vue** | 提供按钮、弹窗、表单、侧边栏、时间轴等现成组件，快速搭建专业界面。 |
| **协同引擎** | **Yjs (CRDT)** | 多人实时共同编辑。自动处理冲突与离线同步，无需手动写锁机制。 |
| **通信方案** | **WebRTC / WebSocket** | `y-webrtc` 实现 P2P 直连（无需服务器）；`y-websocket` 用于有中心服务器的场景。 |
| **本地持久化** | **JSON + IndexedDB** | 数据以 JSON 文件保存本地；`y-indexeddb` 支持离线编辑自动缓存。 |
| **版本控制** | **Rust `git2`** | 后台静默集成 Git，每次保存自动 Commit，实现"时光机"式版本回溯。 |
| **后端逻辑** | **Rust** | 负责文件读写、系统调用、Git 操作。高性能、内存安全。 |
